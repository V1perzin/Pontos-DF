
# Pontos DF - Frontend (prototype)

## Requisitos
- Node.js >= 18
- npm

## Instalar
```bash
cd frontend
npm install
```

## Configurar (opcional)
Crie um arquivo `.env` com:
```
VITE_API_URL=http://localhost:4000
```

## Rodar
```bash
npm run dev
```
